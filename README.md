# Tim Hortons Coffee App ☕

Simple SwiftUI iOS application that allows team members to save and manage their daily Tim Hortons coffee orders.

## Features

- Add customer orders
- Select drink and size
- View saved orders
- Delete orders
- Coffee Run Timer

## How to Use the App

### Add a New Order
- Tap the `+` button
- Enter customer name
- Select a drink
- Select a size
- Tap `Save Order`

### Delete an Order
- Swipe left on an order
- Tap `Delete`

### Use the Coffee Timer
- Tap `Timer`
- Press `Start`
- Use `Pause` or `Reset` when needed
